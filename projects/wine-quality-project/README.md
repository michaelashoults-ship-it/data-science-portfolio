# Wine Quality Project

## Overview
This project was created with a group for STA 570: Basic Statistical Analysis. It uses data based on red and white variants of Portuguese "Vinho Verde" wine. The project explores relationships between physicochemical properties and wine quality. The project applies exploratory data analysis, correlation analysis, multiple linear regression, model selection, and diagnostic testing to identify which variables are most strongly associated with wine quality.

## Dataset
The dataset contains measurements of Portuguese red or white wines and includes physicochemical variables such as acidity, alcohol content, density, pH, sulfates, and other chemical properties. Wine quality serves as the response variable.

The data comes from the UC Irvine Machine Learning Repository: https://archive.ics.uci.edu/dataset/186/wine+quality
